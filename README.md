# FP-Mod-Fast-Ships-and-Signals

This is a developer mod for the narrative strategy god sim "The Fermi Paradox"

https://fermi-paradox.com/

For a faster game the speed of all ships and signals is tripled.

Just imagine that the galaxy is much smaller and the starships and radiosignals travel shorter distances? 
 

If you want to know more about Fermi Paradox modding go here https://wiki.fermi-paradox.com/doku.php?id=setup
